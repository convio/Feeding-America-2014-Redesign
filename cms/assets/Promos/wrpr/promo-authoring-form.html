Promo Type
Select the type of Promo. This determines the position and behavior of the Promo within the page and shows the applicable fields.
Title
Naming convention: [promo-section] [conditional-groups] for ease of finding and maintaining promos
Administrative use only
File Name
This field determines the URL. It is pre-populated based on Title, but may be edited.
Use lowercase, hyphenated, with keywords. No need to include the ".html." 		.html
Headline
Enter a short Headline of the Promo.
Ex.: Learn, Act, Give, Share, Thank You
Length: 55 char. Ex: Hunger in America exists for over 50 million people.
Length: 25 - 75 chars.
Note: The headline text will appear in small font class="small".
If you desire first row of text to be large enter headline into subtitle & then enter small text in a body & assign class="small" to the HTML element containing text.
Length: 85 chars.
Subtitle
Enter a short Subtitle or Intro text for the Promo.
Length: 78 chars. max. Ex. Give hope to children and families in need year round. Become a monthly donor
Length: 110 char. - 270 char.
Note: The subtitle text will appear in big font size class="big".
Promo Date
Select the date of the Promo, to be used in determining the most recent Promo for each Promo type. 
Promo Conditional Group
Select the constituent group(s) or audience for the Promo, as a basis for conditional content.
Note: The promos will only appear if the selected categories match categories defined in the promo matrix.
If this promo applies to two different buckets, you may select all the applicable categories for those buckets to have the same promo appear under those conditions. Ex: If you want the same promo to appear for anonymous users and for users who are logged in but not donors select Default, Logged In, Not a Donor.
Promo Section
Select the Section/Topic of the site to determine the Sections where the Promo should appear. This is the default dynamic behavior site-wide. There are 2 ways to override the dynamic display of Promos by Section:
1. Select "Global".  This will push the Promo out site-wide to its designated position (based on Promo Type). 
2. You can also select Promos manually in the Page properties for the individual Page, which overrides both Global and Section settings. Conditionals within the Promo itself also determine contents.  See promo matrix for details.
Global?
Select yes if the Promo should appear globally at a given time. Change to no if the Promo should no longer display globally. This will override the Promo Section.
Client Image
Select "yes" if the Promo contains a Client Image.
Food Bank
Select a related Food Bank for the promo, if applicable.

Promo Background Color
Select the background color of the Promo: 
- red: class="red" #b22d3a
- dark green: class="green" #53682b
- blue:  class="blue" #006b9d
- purple: class="purple" #5a2559
- orange: class="orange" #e98300
- grey: class="light-gray" #fafafa
- white: transparent, or class="white" #ffffff
For Section Landing pages only:
- orange with wheatstalks
- green with wheatstalks
Default: If no color is selected transparent will be used
Action Button Color
Select the color for the button.
- red: class="button red" #b22d3a
- dark green: class=" button green" #53682b
- blue: class=" button blue" #006b9d
- purple:    class=" button purple" #5a2559
- orange: class=" button orange" #e98300
Default: If no color is selected orange will be used
Action Button Style
Select the shape of the button: square without border, or seed with white border.
Default: If no style is selected, square will be used.
Action button Label
Enter the text that should appear in the Call to Action button, such as "Take Action," "Learn More," or "Donate Now," etc.
Default: If no label is entered the button will not display
If the Promo/Action button URL field is filled in, but not the label, all promo elements will link to it and button will not display.
Promo / Action button URL
Enter the URL that the button or the entire Promo should link to
Default: If URL is not entered the promo will not be linked and the button will not appear
